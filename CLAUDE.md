# CLAUDE.md — Learnify AI

This file is the definitive onboarding guide for developing on Learnify AI. It documents
every file in the repository, the architecture, the main workflows, environment
variables, local dev setup, and conventions to follow when adding new features.

> **Product**: Learnify AI turns a single topic prompt into a full, structured course —
> modules, lessons, embedded videos, visual diagrams, an AI tutor, Hinglish narration,
> and module quizzes — generated by a crew of LLM agents and tracked with a
> gamified XP/gold/streak progress system.

---

## 1. Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React 19, React Router 7, Tailwind CSS 4, Framer Motion, Vite 8 |
| Backend | FastAPI, Motor (async MongoDB driver), Pydantic v2, CrewAI |
| Database | MongoDB (local via Docker Compose, or Atlas in production) |
| LLMs | OpenAI (`gpt-4o-mini`, default) or Gemini — switchable via `LLM_PROVIDER` |
| Auth | Clerk (session JWT verification via JWKS; Google sign-in etc. handled by Clerk) |
| Job queue | MongoDB-backed (no Redis/Celery) — polling `generation_jobs` collection |
| Deployment | Vercel (frontend) · Render (backend) · MongoDB Atlas (database) |

# AI Agent Search & Execution Protocol

1. **PRIMARY NAVIGATION (The Map):** Always consult the Project Architecture tree above first. Treat it as your primary manual for locating core domains and logic. 

2. **FALLBACK SEARCH (The Scalpel):** If the exact file or logic is NOT clearly identifiable in the tree, you are permitted to use file discovery commands, subject to these strict limits:
   - **Targeted Grepping:** Search for exact function names, variable names, or highly unique strings. 
   - **Scope restriction:** Always restrict your search to the most likely subdirectory (e.g., `grep -rnw "backend/app/services" -e "award_rewards"` rather than searching the whole project).

3. **MAINTAIN THE MAP (Self-Documentation):** If your tasks require you to create a new file, folder, or significant module, you MUST immediately update the Project Architecture tree in this `CLAUDE.md` file. Add the new path to the tree along with a brief, 1-line annotation explaining its purpose.
